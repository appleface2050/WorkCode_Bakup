春节公告

过年啦过年啦！

嗅嗅全体员工恭祝各位嗅客猴年吉祥，健康呼吸!

一起回家过年啦！春节期间，嗅嗅团队温馨提示您:

自2016年1月31日至2月15日期间，快递小哥们都回家过年啦， 嗅嗅商城只能接受您下单购买，不能发货，同时暂停受理退换货业务， 嗅客们请悉知！2月16日开始，恢复正常，谢谢支持！
自2月1日起至2月15日，客服宝贝们无法在线陪伴大家， 考虑到大家使用中的疑问，我们总结了一些常见问题解答及排查流程， 请您关注嗅嗅官方微信服务号“嗅嗅科技”—点击右下方“客服机器人”， 获取详细信息哦！
嗅嗅温馨提示您，春节期间燃放烟花爆竹会在空气中产生PM2.5\PM10等 颗粒物，可以使用嗅嗅随身检测数值，远离颗粒物呼吸好空气！
