usage: filterlogs.py [-h] [-l L] [-r R] [-o O] [-k K] [-deploy] [-j] [-d]
                     [-maxthread MAXTHREAD] [-maxlog MAXLOG]

Automtic filter log files.

optional arguments:
  -h, --help            show this help message and exit
  -l L, -logpath L      full path/file of local log file (default: )
  -r R, -result R       full file of result file (default: )
  -o O, -out O          full file of out result file (default: )
  -k K, -keywords K     full file of keywords file (default: .\keywords.xml)
  -deploy               whether is deploy tool log, just one log file (default: False)
  -j, -justdownload     just download log, do not query it (default: False)
  -d, -delete           delete the log files that had matched in keyword (default: False)
  -maxthread MAXTHREAD
                        Max threads (default: 10)
  -maxlog MAXLOG        Max handle log files, if it is zero, handle all logs (default: 15)

sample1:  local logs
py filterlogs.py -l="c:\logs" -o="c:\result\outResult.csv"

sample2: deploy logs(need download logs)
py filterlogs.py -deploy -r="c:\deploy_result.csv" -o="c:\result\outResult.csv"

sample3: not deploy logs(need download logs)
py filterlogs.py -r="c:\app_result.csv" -o="c:\result\outResult.csv"

sample4: delete matach logs, after search logs
py filterlogs.py -d -r="c:\app_result.csv" -o="c:\result\outResult.csv"

sample5: just download logs
py filterlogs.py -j -r="c:\app_result.csv" -o="c:\result\outResult.csv"

sample6:  set max thead and max logs
py filterlogs.py -maxthread=30 -maxlog=1000 -l="c:\logs" -o="c:\result\outResult.csv"
