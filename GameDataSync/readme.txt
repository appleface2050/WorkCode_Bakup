简介：
    脚本 包含2个数据源的同步操作，game9 和360，第一次均为全量同步，第二次开始为增量同步。
    
使用环境：mysql + python 2.7

运行： python.exe main.py


结构说明：
    -Code： 源码目录
        -logs   日志目录
        -common 工具操作
        -config 配置操作
        -dal    数据库操作
        -providers  接口数据源操作
    
    -Doc:   相关参考文档目录

其他：
    *.synctime ：接口同步时间记录文件，只保存上次同步操作的时间戳，文件名使用数据源提供商名称
